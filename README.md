# hw07
胸部X光肺炎分类实验报告
 
1. 数据集统计
 
本次实验使用 Chest X-Ray Images (Pneumonia) 数据集，包含儿童胸部X光影像，共 5800 余张。数据集分为 train、test、val 三个文件夹，其中原始 val 样本过少，因此本次实验从 train 集中按 8:2 重新划分训练集与验证集。
 
- 训练集：4173 张
​
- 验证集：1043 张
​
- 测试集：624 张
​
- 类别分布：NORMAL 约占 27%，PNEUMONIA 约占 73%，存在明显类别不平衡。
 
2. 模型结构
 
本次任务采用自行搭建 CNN 模型进行肺炎二分类任务。网络由三层卷积块、池化层、全连接层与 Dropout 层组成，结构简单、训练速度快，适合本次基础分类任务。
 
- 卷积层：32 → 64 → 128 通道，使用 3×3 卷积核
​
- 池化层：每层卷积后均使用最大池化
​
- 全连接层：128 维隐藏层
​
- Dropout：0.5，用于抑制过拟合
​
- 输出层：1 个神经元，sigmoid 激活，用于二分类
 
3. 超参数
 
- 图像尺寸：150 × 150
​
- Batch size：32
​
- Epochs：10
​
- 学习率：0.001
​
- 优化器：Adam
​
- 损失函数：binary_crossentropy
​
- 数据增强：随机旋转、平移、缩放、水平翻转
 
4. 训练结果
 
4.1 训练曲线
 
模型训练过程中训练准确率稳步上升，验证准确率保持稳定；训练损失持续下降，验证损失后期略有波动，说明模型整体拟合正常，未出现严重过拟合。
 
4.2 混淆矩阵
 
测试集中大部分样本分类正确，肺炎样本识别效果较好；少数正常样本被误判为肺炎，整体分类效果满足医学筛查的基本要求。
 
5. 测试集指标
 
- Accuracy：0.89
​
- Precision：0.88
​
- Recall：0.95
​
- F1 Score：0.91
 
6. 结果分析（任务二）
 
1. 本次实验出现高准确率、高召回率的情况。在医学诊断场景中，召回率更为重要，因为漏诊（假阴性）会导致肺炎患者延误治疗，风险极大。相比之下，误诊（假阳性）仅需进一步复查，风险较低。
​
2. 数据增强明显缓解了过拟合，提高模型泛化能力；迁移学习（如 ResNet50）能提取更强特征，进一步提升精度，尤其适合医学图像这种特征复杂的场景。
​
3. 若模型将肺炎误判为正常（假阴性），会导致患者错过最佳治疗时机，甚至病情恶化；若将正常误判为肺炎（假阳性），会增加患者心理压力与不必要的医疗成本。
